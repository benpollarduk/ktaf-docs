Paragraph

class Paragraph(val line: String, val delta: Int = 1, val action: ConversationAction = { })

A paragraph within a Conversation. Must contain a line, but can also have an optional action that is invoked as a response to this paragraph being triggered. The delta is a relative pointer to the next element within the Conversation.
